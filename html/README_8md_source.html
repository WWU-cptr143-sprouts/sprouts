1 sprouts
2 =======
3 Every spring quarter the CPTR 143 programming class at Walla Walla University
4 codes Conway's Game of Sprouts for a final project. The students are supposed
5 to create a new version from previous code with the final goal of creating an
6 unbeatable AI. However, except for one three-year streak, most years people
7 recode the program basically from scratch due to differing design approaches,
8 not wanting to read other people's messy code, etc. The CPTR 143 class of 2013
9 chose to take the 2010v2 code, which looked like a single person's side project
10 to check for line crossings graphically, as the basis for the new version.
11 However, like other years, after spending much time in designing a foundation,
12 a simple and unified data structure for storing nodes and lines, we chose to
13 recode most of the basic design. We have provided the A-Checker as a working,
14 well-tested foundation for future years and provided an experimental,
15 partially-working AI and GUI that respectively inherit from each other and
16 from Game, the A-Checker.

18 Using
19 -----
20 Because of people's differing preferred platforms, we have made this
21 cross-platform.
22 
23 ### Windows ###
24 Throughout the year we used CodeBlocks to code in class. We chose to provide
25 project files for Windows in ``codeblocks/Sprouts_windows.cbp``. This will
26 link to the included SDL libraries. If you want to run the A-Checker tests,
27 build the project ``codeblocks/AChecker.cbp``.
28 
29 **Note:** On Windows SDL will not output to the command prompt, so you'll
30 have to open ``stdout.txt`` to see the output.
31 
32 ### Mac ###
33 To install the libraries, you can use any package manager, for example:
34 **Fink** *(untested)*: ``fink install sdl sdl-image sdl-gfx13 sdl-ttf``
35 **Macports** *(tested)*: ``port install libsdl libsdl-framework libsdl_gfx libsdl_image libsdl_ttf``
36 
37 After this, you can either setup a CodeBlocks file or run ``make`` to compile
38 and ``./sprouts`` to run. To run the tests, ``make tests; ./tests/tests``.
39 
40 ### Linux ###
41 Install whatever provides the following. Dependencies:
42 
43  * SDL
44  * SDL\_gfx
45  * SDL\_image
46  * SDL\_ttf
47 
48 Then, run ``make`` to compile and ``./sprouts`` to run. You can also run the
49 tests with ``make tests; ./tests/tests``.

51 Project Files
52 -------------
53 To provide easy access to all of the files, they are stored in two places. The
54 code can be accessed from the GitHub link below. If future years decide to use
55 this code as a basis, they can easily either fork it or clone and push to their
56 own repository on GitHub or elsewhere. A few key documents are in the *docs/*
57 folder, but the rest of the documentation is on the wiki linked to below. All
58 of the classes have a page documenting what they consist of for easy reference
